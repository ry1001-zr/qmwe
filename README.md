# 课程管理系统

一个基于HTML、CSS和JavaScript开发的静态课程管理系统，提供完整的课程展示、筛选、详情查看和用户注册功能。

## 项目概述

本系统是一个响应式的课程管理平台，无需后端支持，适合作为课程设计项目。主要功能包括：

- 🏠 **首页轮播图**：展示热门课程推荐
- 📚 **课程列表**：支持多条件筛选和搜索
- 🔍 **课程详情**：详细的课程信息和评价展示
- 📝 **用户注册**：完善的表单验证和交互体验
- 🎨 **动画效果**：丰富的页面交互动画

## 页面功能

### 1. 首页 (index.html)
- **轮播图功能**：自动播放、手动切换、触摸滑动
- **热门课程展示**：动态加载、悬停效果
- **统计数据动画**：滚动触发、数字递增
- **整体响应式设计**：移动端适配

### 2. 课程列表页 (courses.html)
- **多维筛选**：按分类、难度、价格筛选
- **实时搜索**：关键词搜索，包含防抖功能
- **排序功能**：按价格、评分、热门度排序
- **分页加载**：加载更多按钮实现
- **筛选标签**：显示当前筛选条件

### 3. 课程详情页 (course-detail.html)
- **图片画廊**：缩略图预览、大图展示
- **标签页切换**：概述、课程大纲、讲师介绍、评价的懒加载
- **视频预览**：免费课程试看功能
- **收藏功能**：本地存储收藏状态
- **购买流程**：模拟支付过程
- **相关课程推荐**：基于分类的智能推荐

### 4. 注册页面 (register.html)
- **实时表单验证**：输入时即时反馈
- **密码强度指示**：动态显示密码强度
- **验证码刷新**：点击刷新验证码
- **兴趣爱好选择**：多选框交互
- **响应式反馈**：错误提示、成功动画
